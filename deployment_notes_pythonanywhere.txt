# Setup description for PythonAnywhere:
https://www.pythonanywhere.com/

1. create a virtual environment in console
mkvirtualenv myvirtualenv --python=/usr/bin/python3.9

2. open the virtual environment
workon myvirtualenv

3. install packages from mysite
pip3.9 install -r requirements.txt

4. change working directory and path of the virtual environment in web-tab:
/home/BBeuster/
/home/BBEUSTER/.virtualenvs/myvirtualenv

5. Define WSGI configuration file:
/var/www/bbeuster_pythonanywhere_com_wsgi.py
